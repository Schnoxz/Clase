--> Use for when the number of iterations is known

--> Use while when you loop based on a condition that may change

--> Use do...while when the code must run at least once


for example	:		for (int i = 0; i < 5; i++) {
					System.out.println("Hello " + i);
					}
				
				
				
while example :		int i = 0;
					while (i < 5) {
					System.out.println("Hello " + i);
					i++;
					}



do while example :  int i = 0;
					do {
					System.out.println(i);
					i++;
					} while (i < 5);



					int[] numbers = {5, 10, 15, 20};
					int total = 0; // accumulator

					for (int num : numbers) {
					total += num; // add each number to total
					}

					System.out.println("Total sum: " + total);
